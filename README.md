# Exploring Airbnb Market Trends

## Project Overview

This project explores Airbnb listing data to uncover trends in pricing, room types, and customer reviews. Using Python and the Pandas library, I cleaned, merged, and analyzed data from multiple sources to answer key questions about the Airbnb market.

This project was completed as part of the **DataCamp Data Analyst in Python** learning path.

---

## Objectives

- Merge multiple datasets into a single DataFrame.
- Clean and preprocess the data.
- Analyze Airbnb listing prices.
- Explore the distribution of room types.
- Identify the earliest and most recent customer reviews.
- Produce summary statistics and insights.

---

## Technologies Used

- Python 3
- Pandas
- NumPy
- Jupyter Notebook

---

## Skills Demonstrated
- Reading CSV, Excel, and TSV files with Pandas
- Merging multiple DataFrames
- Working with datetime data
- Performing exploratory data analysis (EDA)
- Calculating descriptive statistics
- Presenting insights with clean, readable Python code
